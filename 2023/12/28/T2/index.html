一、（关不掉）的重复显示窗口

​ 在电脑桌面新建一个TXT文本。在txt文件里里，输入msgbox“XXX”（引号里面就是弹窗会出现的内容）。要记得do，loop一定要写！关不掉的原因就是它。

do
msgbox”不好，电脑中毒了，是否关闭”
msgbox”关闭失败”
msgbox”是否重试”
msgbox”关闭失败，即将重启，是否取消”
msgbox”取消失败，即将关机”
msgbox”关机还有10秒”
msgbox”无法关机”
loop

​ 文件名任意写，尽量起一些会让人好奇点击的名字。记得扩展名写vbs/vbe，保存类型也要改成所有文件。写完后可以另存为

双击打开，来看效果怎样

如果想要关掉，用任务管理器强制关掉就可以啦。

如果有防火墙的话是打不开的，会被拦下。

二、轰炸屏幕的炸弹窗口（慎用）

新建一个TXT文本，然后在文本里写入程序。

:start

start cmd

goto start

文件名起一些会让人好奇点击的名字。记得扩展名改写成.bat，保存类型也要改成所有文件。

然后当你打开这个文件时，你的桌面会疯狂地弹出窗口，就像炸弹一样轰炸你的电脑，严重的可能会死机，所以这个自己慎用哈。
